  - ### **15.3 When to Apply Binary Search?**  
    - **Array is Sorted** – Works only on sorted data.  
    - **Search Problem** – Finding an element's index efficiently.  
    - **Optimization Problems** – Finding min/max solutions (e.g., min capacity, peak element).  
    - **Range-Based Problems** – Searching for first/last occurrence.  
    - **Monotonic Functions** – Problems with increasing/decreasing patterns.  
    - **Time Complexity:** `O(log n)` (Fast for large datasets).  
    - **Avoid if Unsorted Data** – Use Linear Search instead.
